# Security Policy

## Supported Versions

The following table lists the versions of Dolphin Emu currently supported with security updates. Only the latest stable and development branches receive security fixes. Older versions are not maintained for security.

| Version | Supported          |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :x:                |
| 4.0.x   | :x:                |
| < 4.0   | :x:                |


## Reporting a Vulnerability

To report a security vulnerability in Dolphin Emu, please email security@dolphin-emu.org with details of the issue, including steps to reproduce and any relevant information. Alternatively, you may open a private security advisory via GitHub.

We aim to acknowledge receipt of all vulnerability reports within 5 business days. You will receive updates on the status of your report as we investigate and address the issue. If the vulnerability is confirmed, we will work with you on disclosure timing and credit. If the report is declined, we will provide an explanation.
